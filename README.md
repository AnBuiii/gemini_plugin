## Description

## Installation

- Using the IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Gemini
  Plugin"</kbd> >
  <kbd>Install</kbd>

- Manually:
